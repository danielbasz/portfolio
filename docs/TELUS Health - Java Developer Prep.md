## Technical Interview Prep Plan

### Priority 1: Core Java (Must Know Cold)

- [ ] **OOP Fundamentals** - Classes, inheritance, polymorphism, encapsulation, abstraction
- [ ] **Collections Framework** - List, Set, Map implementations, when to use which
- [ ] **Exception Handling** - Checked vs unchecked, try-with-resources, custom exceptions
- [ ] **Generics** - Type parameters, bounded types, wildcards
- [ ] **Streams & Lambdas** - Functional interfaces, stream operations, collectors
- [ ] **Concurrency Basics** - Threads, synchronized, ExecutorService, CompletableFuture
- [ ] **Java 8+ Features** - Optional, new Date/Time API, default methods
- [ ] **Memory Management** - Stack vs heap, garbage collection basics

### Priority 2: Spring Boot / Spring Framework

- [ ] **Dependency Injection** - @Autowired, @Component, @Service, @Repository
- [ ] **Spring Boot Basics** - Application structure, auto-configuration, properties
- [ ] **REST Controllers** - @RestController, @RequestMapping, @GetMapping, @PostMapping
- [ ] **Exception Handling** - @ControllerAdvice, @ExceptionHandler
- [ ] **Data Access** - Spring Data JPA, repositories, query methods
- [ ] **Testing** - @SpringBootTest, MockMvc, @MockBean

### Priority 3: Swing UI (Their Legacy Stack)

- [ ] **Component Hierarchy** - JFrame, JPanel, JComponent
- [ ] **Layout Managers** - BorderLayout, GridLayout, GridBagLayout, BoxLayout
- [ ] **Event Handling** - ActionListener, MouseListener, event dispatch thread
- [ ] **Threading in Swing** - SwingUtilities.invokeLater, SwingWorker
- [ ] **Common Components** - JTable, JTree, JList, JTextField, JComboBox
- [ ] **MVC in Swing** - Models, renderers, editors

### Priority 4: Databases (Oracle/PostgreSQL)

- [ ] **SQL Fundamentals** - SELECT, JOIN types, GROUP BY, HAVING, subqueries
- [ ] **Indexes** - When to use, B-tree vs others, composite indexes
- [ ] **Transactions** - ACID properties, isolation levels
- [ ] **JDBC** - Connection, PreparedStatement, ResultSet, connection pooling
- [ ] **JPA/Hibernate** - Entities, relationships, lazy vs eager loading, N+1 problem
- [ ] **Query Optimization** - EXPLAIN plans, avoiding full table scans

### Priority 5: Cloud & Modernization

- [ ] **Cloud Concepts** - IaaS, PaaS, SaaS, 12-factor apps
- [ ] **GCP Basics** - Compute Engine, Cloud SQL, Cloud Storage, Cloud Run
- [ ] **Containerization** - Docker basics, Dockerfile, docker-compose
- [ ] **Microservices** - Patterns, API gateway, service discovery concepts
- [ ] **CI/CD** - Jenkins pipelines, automated testing, deployment strategies

### Priority 6: Supporting Technologies

- [ ] **XML/XSLT** - Parsing (DOM, SAX, StAX), XPath, transformations
- [ ] **Testing** - JUnit 5, Mockito, integration testing strategies
- [ ] **Git** - Branching strategies, rebasing, merge conflicts
- [ ] **Design Patterns** - Singleton, Factory, Builder, Observer, Strategy
